Cally Study Assistant

There are two ways to run Cally Study Assistant.
 
Option 1: Run the executable
https://drive.google.com/file/d/1-0M-D_CoQBpZHpbEsQjsgmNbxTFmvCWg/view?usp=sharing
Double-click `Cally.exe`. The build is self-contained, so all dependencies are bundled: no extra installs are going to be needed (Windows 10 build 17763 / Windows 11 or newer).

Option 2: Open in Visual Studio 2022

1. Open `Cally Calendar.sln`.
2. On first load, let NuGet restore the package references. Visual Studio does this automatically; if it doesn't, right-click the solution in Solution Explorer and choose "Restore NuGet Packages".
3. At the top of the window, set the launch profile dropdown (next to the green play arrow) to "Cally Calendar (Unpackaged)".
4. Click the green play button.
